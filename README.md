# BBQ-Escape-game

## Description
BigBaQyard Escape is a game where the player (a burger) has to avoid to touch with the enemy (a hungry farmer). The player gets points for the played time. After touching the bonus (a ketchup≈õ's bottle) the player gets extra points. The game has no end. The player has 2 lives. After 1st touch of the enemy player continue the game, after 2nd the game is over
